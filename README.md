# justLanded

A local co-op 3D physics game built in Godot 4. Players are stranded astronauts scavenging debris on a tropical island. Scrap together a janky vehicle using scrounged parts (wheels, thrusters, scrap metal) and escape before the vehicle falls apart. Each run randomizes the control bindings per seed, forcing players to adapt. Low-poly aesthetic, physics-driven gameplay, chaos guaranteed.
